# EchoMind App Frontend

Flutter frontend project for EchoMind AI Error Tracker.

## Quick Start

### 1) Requirements

- Flutter `>= 3.27.0`
- Dart `>= 3.6.0` (included with Flutter SDK)

Check your environment:

```powershell
flutter doctor
```

### 2) Install dependencies

Run from this project directory (the same folder as this `README.md`):

```powershell
flutter pub get
```

### 3) Start the frontend

Recommended (Web):

```powershell
flutter run -d chrome
```

Windows desktop:

```powershell
flutter run -d windows
```

If you are unsure which targets are available:

```powershell
flutter devices
```

## Useful commands

```powershell
flutter clean
flutter pub get
flutter test
```
